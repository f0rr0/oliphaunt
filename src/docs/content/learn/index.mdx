# Learn

Use these pages after the first query works and you need to make production
choices: where data lives, which runtime boundary fits, how mobile lifecycle
behaves, and how to package the app without extra extension files.

## What Learn Covers

Learn pages explain product behavior in app terms. Use the SDK page when you
need install steps and code. Use Reference when you need a matrix, catalog, or
version lookup.

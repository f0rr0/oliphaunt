Oliphaunt is embedded PostgreSQL for apps. Install the SDK for your app target,
open a PostgreSQL root inside app-owned storage, run SQL, and select only the
extensions your app ships.

This page is the shortest tutorial path. Pick one app target, prove the runtime
with one query, then move into the SDK, runtime, extension, or storage page that
matches the next decision.

## First Query Shape

Every SDK uses ecosystem-native syntax. The application flow stays the same:
choose storage, select a runtime mode, select exact extensions, open, query,
and close or detach through the SDK lifecycle.

# @oliphaunt/broker-win32-x64-msvc

Windows x64 MSVC broker helper package for `@oliphaunt/ts`. This package is
populated by Oliphaunt release packaging.

The package includes only the helper's import-derived app-local Microsoft Visual
C++ runtime closure, with a SHA-256 receipt. The unmodified DLLs remain subject
to the [Microsoft Visual Studio redistribution
terms](https://learn.microsoft.com/visualstudio/releases/2022/redistribution).

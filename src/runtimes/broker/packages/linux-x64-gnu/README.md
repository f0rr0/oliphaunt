# @oliphaunt/broker-linux-x64-gnu

Linux x64 glibc broker helper package for `@oliphaunt/ts`. This package is
populated by Oliphaunt release packaging.

The complete Oliphaunt Linux GNU surface is qualified at glibc 2.38 with a
`GLIBCXX_3.4.30` ceiling. This package name identifies an ABI, not every Linux
distribution.

# @oliphaunt/liboliphaunt-win32-x64-msvc

Windows x64 MSVC native library package for `@oliphaunt/ts`. This package is
populated by Oliphaunt release packaging.

The package includes the exact app-local Microsoft Visual C++ runtime provider
closure required by the native runtime and its supported extensions, with a
SHA-256 receipt. The unmodified DLLs remain subject to the [Microsoft Visual
Studio redistribution terms](https://learn.microsoft.com/visualstudio/releases/2022/redistribution).

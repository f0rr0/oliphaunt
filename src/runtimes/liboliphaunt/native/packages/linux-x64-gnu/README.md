# @oliphaunt/liboliphaunt-linux-x64-gnu

Linux x64 glibc native library package for `@oliphaunt/ts`. This package is
populated by Oliphaunt release packaging.

Requires glibc 2.38 or newer and a GNU C++ runtime providing
`GLIBCXX_3.4.30`. This package name identifies an ABI, not every Linux
distribution.
